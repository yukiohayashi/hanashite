## 問題点のまとめ

### 1. 「夫婦」カテゴリの問題
- **漠然としている**: 夫婦関係全般を指すため、具体的な悩みが分かりにくい
- **恋愛掲示板に不適切**: ハナシテは「恋愛相談」がコンセプトなので、既婚者の夫婦関係は対象外
- **削除推奨**: 夫婦関係の悩みは「結婚生活」や「マンネリ」などより具体的なカテゴリに分割

### 2. 重複・統合可能なカテゴリ
- **「性」と「セックスレス」**: 「夜の悩み」に統合
- **「格差恋愛」**: ニッチすぎる、「その他」に統合

### 3. 削除検討カテゴリ
- **「離婚」**: 恋愛掲示板に不適切（既婚者の離婚相談は対象外）

### 12個のカテゴリが最適な理由
